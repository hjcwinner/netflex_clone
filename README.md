## 주요기능
----------
* API 연동 및 받아온 정보 뿌려주기
* 인기영화, 상영영화, 영화세부내용 etc. 구현
* Movie, TV Show 메인화면 구현 (Popular, ComingSoon, Top Rated 영역)
* 검색페이지 구현, 검색 후 Movie, Tv Results 키워드 서칭에 맞게 결과 출력
